Mi proyecto consiste en el desarrollo de un sitio web personal como trabajo individual de la asignatura Fundamentos de la Ingeniería Informática. El sitio presenta información académica y contenidos relacionados con el grado que estoy cursando.

El sitio está formado por varias páginas HTML enlazadas entre sí (inicio, información personal, grado, asignatura FII, tema, red y contacto), todas ellas con un diseño común.

He utilizado HTML para la estructura y CSS para el diseño visual, manteniendo un estilo sencillo y uniforme. No se han empleado scripts ni frameworks externos.

El proyecto se ha gestionado con Git y publicado mediante GitHub Pages, realizando commits progresivos durante el desarrollo.

Este trabajo ha permitido aplicar los conocimientos básicos de desarrollo web y control de versiones que aprendi en la asignatura.
